# 멕시코 딥페이크 관련 법 요약

## 핵심 법령/제도
- `Federal Criminal Code AI intimate-content bill`은 인공지능을 이용한 사적/친밀 콘텐츠 생성 문제를 다루기 위해 Federal Criminal Code 제199 Octies 추가를 제안하는 문서입니다. 현재 코퍼스 기준으로는 시행법이 아니라 발의/제안 단계로 취급합니다. (Ref1)
- `Labor/Copyright performers AI-rights reform`은 2026년 5월 14일 Diario Oficial de la Federacion에 공포된 개정으로, 예술가·실연자의 이미지 또는 음성을 AI 시스템 등으로 사용하는 경우 노동계약에 조건과 보수를 구체적으로 명시하도록 합니다. (Ref2, Ref3)

## 요약
- 멕시코 자료는 두 축으로 나뉩니다. 하나는 AI 생성 친밀 콘텐츠에 대한 형사법 발의이고, 다른 하나는 실연자 이미지/음성의 AI 사용에 관한 노동·저작권 보호입니다. (Ref1, Ref2, Ref3)
- 시행 중인 축은 일반 딥페이크법은 아니지만, 합성미디어 제작에서 초상·음성·동의·보수 문제와 직접 연결됩니다. (Ref2, Ref3)

## Policy Dimensions
- `라벨링(표시) 법제화 상태`: 법률 없음: 현재 멕시코 코퍼스에서 일반 딥페이크 라벨링 의무를 정한 전용 법률은 확인되지 않습니다. 실연자 권리 개정은 공개 표시보다 계약상 동의·조건·보수 명시에 초점이 있습니다. (Ref2, Ref3)
- `선거 관련 법제화 상태`: 법률 없음: 현재 멕시코 파일에서는 선거 딥페이크 전용 규정이 확인되지 않습니다. (Ref1, Ref2, Ref3)
- `동의 없는 딥페이크 법제화 상태`: 형사법 발의문은 AI로 생성된 친밀 콘텐츠를 직접 겨냥하지만, 현재 코퍼스에서는 제안 단계입니다. 시행 중인 노동·저작권 개정은 실연자의 이미지/음성 사용에 대한 동의 기반 통제를 강화합니다. (Ref1, Ref2, Ref3)
- `서비스 제공자 및 플랫폼 책임 법제화 상태`: 법률 없음: 현재 파일만으로는 딥페이크 서비스 제공자/플랫폼 전용 의무체계가 확인되지 않습니다. (Ref1, Ref2, Ref3)
- `처벌 법제화 상태`: 제안 단계/약함: AI 친밀 콘텐츠 형사처벌은 Federal Criminal Code 개정 발의와 연결되어 있으므로, 공식 시행 근거가 추가되기 전까지 proposed로 표시해야 합니다. (Ref1)

Ref1: `sources/official/mexico/2_draft/mexico-2026-federal-criminal-code-ai-intimate-content-bill-gazette.pdf`
Ref2: `sources/official/mexico/1_official-law-act/mexico-2026-labor-copyright-performers-ai-rights-dof-decree.pdf`
Ref3: `sources/official/mexico/1_official-law-act/mexico-2026-federal-copyright-law-current-lfda.pdf`

## 실무 포인트 (Dimension별)
- `라벨링(표시)`: 일반 표시 의무가 확인되기 전까지는 내부 정책 기준으로 공개/고지 통제를 운영합니다.
- `선거`: 선거 맥락 합성콘텐츠는 추가 공식 선거법 자료 확보 전까지 고위험 검토 대상으로 분류합니다.
- `동의 없는 딥페이크`: 제안 단계 형사책임과 시행 중인 실연자 이미지/음성 동의권을 구분해 검토합니다.
- `서비스 제공자/플랫폼 책임`: 현 코퍼스만으로 멕시코 전용 플랫폼 책임체계를 과장하지 않고, 신고-조치 기록은 내부 준수 체계로 유지합니다.
- `처벌`: 형사처벌 수위는 후속 DOF 또는 현행 형법 조문 확인 전까지 proposed로 유지합니다.

## 확인 필요
- 2026년 3월 23일 Gazette의 Federal Criminal Code AI intimate-content proposal이 이후 통과·공포되었는지 확인해야 합니다.
- 멕시코의 선거·플랫폼·개인정보·초상권 관련 추가 공식 법령을 더 수집해야 합니다.
