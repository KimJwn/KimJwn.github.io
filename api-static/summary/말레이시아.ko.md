# 말레이시아 딥페이크 관련 법 요약

## 핵심 법령/제도
- 말레이시아는 현재 작업 코퍼스에서 수집 중 / 관련 법 부재로 표시합니다. 확인된 딥페이크 전용 공식 원문 파일은 아직 추가되지 않았습니다.
- 이 항목은 18개 jurisdiction 네비게이션에는 포함하되, 확인된 공식 근거와 수집 대기 상태를 명확히 분리하기 위해 유지합니다.

## 요약
- 말레이시아는 현재 공식 원문 코퍼스 기준으로 수집 중 / 관련 법 부재 상태입니다.
- 공식 문서가 추가되기 전까지 딥페이크 전용 규제 경로를 추론하지 않습니다.

## Policy Dimensions
- `라벨링(표시) 법제화 상태`: 법률 없음: 현재 코퍼스에는 일반 딥페이크 라벨링 또는 워터마킹 의무를 확인하는 공식 원문이 없습니다.
- `선거 관련 법제화 상태`: 법률 없음: 현재 코퍼스에는 선거 딥페이크 공식 원문이 없습니다.
- `동의 없는 딥페이크 법제화 상태`: 법률 없음: 현재 코퍼스에는 확인된 NCII 딥페이크 공식 원문이 없습니다.
- `서비스 제공자 및 플랫폼 책임 법제화 상태`: 법률 없음: 현재 코퍼스에는 딥페이크 서비스 관련 플랫폼 책임 공식 원문이 없습니다.
- `처벌 법제화 상태`: 법률 없음: 현재 코퍼스에는 확인된 처벌 근거 공식 원문이 없습니다.

## 수집 상태
- 수집 중 / 현재 코퍼스 기준 관련 법 부재.
