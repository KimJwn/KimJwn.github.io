# Hidden Landing Page

This folder is meant to occupy `/deepfake-policy/` on GitHub Pages with a non-public placeholder page.

## Intent

- Do not link this path from the public profile homepage.
- Show a private-preview notice instead of the working app.
- Reduce search indexing with `noindex`.

## Important

GitHub Pages cannot enforce real access control. If true restriction is required, the project must be hosted behind authentication on another platform.
